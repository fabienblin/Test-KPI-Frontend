Technical Solution


Frontend

The frontend of this website is built with Angular 13 on top of Node 17 and uses Bootstrap 5 because it's the best CSS framework.

It is deployed and hosted on Heroku from it's Github sources.

The Angular router comunicates with the backend REST API via HTTP using the standard methods (GET, PUT, PATCH, DELETE, HEAD, OPTIONS).

Links to the sources :

  • Hosted Link (this very web site)
  • Github repository

Backend

The backend is built using Django 3 on top of Python 3 and uses the rest_framework package to implement the API methods.

It is deployed and hosted on Heroku from it's Github sources like the frontend.

This app manages the PostgreSQL database which was populated by a json file with a migration script.

Links to the sources :

  • Hosted Link
